Entity Linking

This early version has various known issues, mainly:

  • No support of fuzzy search.
  • Long runtime due to missing parallelization of queries.
  • Recognized Entities without candidates are not displayed.
  • Output is not deterministic.

If you experience any other problem don't hesitate to open an Issue on github.

This is an very early alpha version of an entity linking system developed as a part of a Knowledge Base Population System in my Master's Thesis. The underlying ideas, especially the optimization algorithm was first published by Alberto Parravicini: “Fast and Accurate Entity Linking via Graph Embedding.”.
